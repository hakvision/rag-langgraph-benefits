이 페이지는 안내용, 실제 인터랙티브 UI는 집 PC에서 로컬로 띄운다

GitHub Pages는 정적 호스팅이라 로컬 8B/14B 백엔드에 직접 붙어 동작하는 UI를 안정적으로 제공하기 어렵다. 그래서 실제 질문 입력/로그 확인 UI는 집 PC에서 직접 서버를 띄워서 쓰는 구조로 만들었다.

실행 방법

cd rag-langgraph-benefits
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python server.py

그 다음 브라우저에서 아래 주소로 접속:

http://127.0.0.1:8000/

무슨 일이 일어나나

  • 네가 질문 입력
  • 8B router가 질문 난이도/종류를 분류
  • 8B로 검색 가능한 질문이면 8B 검색
  • 복잡하거나 실패하면 14B로 승격
  • 검색 품질이 낮으면 같은 노드를 재시작
  • 최종 답변 + 검색 청크 + 상세 로그를 한 화면에 표시

RAG 문서

현재 기본 RAG 문서는 LangGraph 공식 agentic RAG 튜토리얼을 가져오도록 연결해 두었다:

https://docs.langchain.com/oss/python/langgraph/agentic-rag
